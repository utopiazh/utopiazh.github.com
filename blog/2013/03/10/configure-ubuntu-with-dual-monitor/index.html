Configure Ubuntu With Dual Monitor

Mar 10th, 2013

Background

After installing Ubuntu and enabling dual monitor, the default layout is:

Laptop --> LCD Monitor with HDMI

while my placement of monitors are:

LCD Monitor with HDMI --> Laptop

Solution

Searched around the web, and found the issue could be resolved by a simple xrand script:

”’

!/bin/sh

if [ “$1” != “off” ]; then # External Screen as HDMI, Laptop below echo “Dual Screen on” xrandr –output VGA1 –off xrandr –output HDMI1 –primary –mode 1920x1080 xrandr –output LVDS1 –off xrandr –output LVDS1 –mode 1600x900 –right-of HDMI1 else # External Screen off, Laptop primary echo “Dual Screen off” xrandr –output HDMI1 –off xrandr –output LVDS1 –primary –mode 1600x900 fi ”’

Save the script as dual_monitor, and add the directory into $PATH. To stop using dual monitor, run: ”’ dual_monitor off ”’ To start using dual monitor, run: ”’ dual_monitor ”’

Reference
